- Base: `#0F1016`
- Accents: `#1BC5C9`, `#FF6A1F`

## Install

1. Copy the `torrentz-hydra` folder into your `Omarchy/themes/` directory.
2. Set `active = "torrentz-hydra"` in your config.
3. Reload Omarchy to apply the theme.
